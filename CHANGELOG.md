# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-21

### Added
- Initial release of Power BI HTML Visual Templates
- HTML KPI Cards template with dynamic effects and embedded SVG visuals for enhanced data presentation
- HTML Waterfall Visual template for AC vs. SPLY/PY variance comparison analysis
- Installation guide for HTML Content custom visual
- Documentation for both templates
- Ready-to-use .pbix files with implementation instructions

### Features
- Comprehensive README with step-by-step setup instructions
- Multiple reusable HTML/CSS templates for Power BI custom visuals
- Support for DAX integration with HTML visuals
- Template library with links to individual template pages
